# Change Log

All notable changes to the "project-launcher" extension are documented in this file.

## [Unreleased]

- Added tags and collections for saved projects with dedicated commands.
- Added collection grouping in the saved-projects tree section.
- Added selectable sort modes (last accessed, name, type, path).
- Added JSON import/export for saved and history project data.
- Added `.code-workspace` save/open support alongside folder projects.
- Added context actions for open in current/new window, split, and new workspace.
- Added configurable project actions that execute in integrated terminals.
- Added Git metadata rendering (branch, dirty state, latest commit) with caching.
- Added quick-open command for fuzzy navigation across saved and history projects.
- Extended tests for workspace support, metadata, filtering/grouping, and command registration.
